# ISO-TP C Implementation

ISO-TP (ISO 15765-2) protocol implementation in C.

## Features

- Full ISO-TP protocol support
- Configurable parameters
- Minimal dependencies

## Quick Start

### Building

```bash
cmake -B build
cmake --build build
```

### Integration

Include the header file:
```c
#include "isotp.h"
```
